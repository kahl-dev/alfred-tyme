# alfred-tyme

> Alfred 3 workflow quick start and stop Tyme2 records

## Install

```
$ npm install --global alfred-tyme
```

*Requires [Node.js](https://nodejs.org) 4+ and the Alfred [Powerpack](https://www.alfredapp.com/powerpack/).*
